## 共通スタイル

- フォント: 游ゴシック（Yu Gothic）または既定の日本語ゴシック。ヘッダー行は白字・濃色塗り（例 #2F5B7C）・太字
- 各シートとも1行目にシートタイトル（大きめ・太字）、ヘッダー行は固定（freeze panes）
- 列幅は内容に合わせ調整。メモ列は折り返し有効
- プルダウン（データ入力規則）と数式は下記指定どおり

## シート① はじめに

- タイトル「チェックリスト — 「思い出カメラ」反応テスト」
- 本文（セル結合したテキストブロックでよい）:
  - このファイルはテストの記録用です。オペレーターが記入します。
  - 記入するのはシート②〜④だけ。⑤は自動集計なので見るだけです。
  - 記入例（グレーの行）を参考にしてください。
- テスター名の対応表（記入欄）: 「テスター1= ＿＿＿ / テスター2= ＿＿＿ / テスター3= ＿＿＿」の3行（名前記入セルは枠線付き空欄）

## シート② 準備チェック

毎回の通話前チェック。表形式:

- 行（チェック項目）:
  1. 家族側ページに Googleでサインインできた
  2. 画面左下に「📊ログ」ボタンが見える
  3. テスター側の顔が画面に映っている（距離50cmくらい）
  4. テスター側の部屋が静か（テレビ消灯）
  5. 画面録画を開始した
  6. このチェックリストを開いた
- 列: Round1-テスター1 / Round1-テスター2 / Round1-テスター3 / Round2-テスター1 / Round2-テスター2 / Round2-テスター3（計6列）
- 各セルはプルダウン「✓」のみ（空欄=未確認）

## シート③ 実施記録（メイン）

1行 = 台本1本 × テスター1名 × ラウンド。**108行を事前生成**（Round1/2 × テスター1〜3 × 台本18本、この順で並べる）。

列構成:

| 列 | 内容 | 入力方法 |
| --- | --- | --- |
| A ラウンド | 1 or 2 | 事前入力済み |
| B テスター | 1/2/3 | 事前入力済み |
| C 番号 | A1〜A6, B1〜B6, C1〜C3 | 事前入力済み |
| D 場面 | 下の対応表のとおり | 事前入力済み |
| E 正解 | 下の対応表のとおり | 事前入力済み |
| F 実施日 | 日付 | 手入力 |
| G 反応した回数 | 数値（0以上） | 手入力 |
| H 期待どおり？ | ◯ / × | プルダウン（空欄可） |
| I メモ | 自由記入 | 手入力・折り返し |

- 先頭に**記入例のグレー行**を1行（例: ラウンド1・テスター1・A1・…・実施日 7/15・回数 0・◯・「問題なし」。集計対象外にするためラウンド列は「例」と記載)

  - A2 ふつうの雑談（10分） → 反応しない

  - A4 生活音を出す → 反応しない
  - A5 最初のあいさつ（かけ直し） → 反応しない
  - A6 立って戻る → 反応しない
  - A7 小さな物音（ささやき・紙・机） → 反応しない
  - B1 声を出して笑う → 反応する
  - B2 驚く → 反応する
  - B3 怒る → 反応する
  - B4 静かに笑う（声量そのまま） → 反応する
  - B5 ほめ言葉（ふだんの声） → 反応する
  - B6 感極まる → 反応する
  - B7 孫の変顔（オペレーター・無言） → 反応する
